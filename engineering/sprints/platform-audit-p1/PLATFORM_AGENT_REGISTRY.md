# Platform Agent Registry

P1-59 establishes `packages/shared/src/agent-registry.ts` as the canonical
platform-wide identity and capability catalog. It consolidates existing agents
without replacing the narrower three-key runtime orchestration registry.

Registered identities are Steve Success, Michael Magnificent, Ivory,
ScriptMaker, and Admin Recommendations. Daily Success Coach and sponsor-assisted
coaching are Michael-owned capabilities rather than separate autonomous agents.

Each descriptor names its mission, surfaces, owned and excluded responsibilities,
human action owner, implementation source, and activation state. Future entries
must use a valid unique key, cite an approved decision, identify their behavior
source, and enter inactive.

Prompt/template versions, inputs, outputs, tests, and degradation behavior belong
to P1-60 and are intentionally not embedded here.
